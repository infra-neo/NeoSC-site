  2. Crear VM Windows (Libcloud/OVH LXD)
  3. Esperar VM ready (poll)
  4. Habilitar WinRM + OpenSSH
  5. Instalar Netbird agent
  6. Registrar peer en NeoMesh
  7. Instalar + activar TSplus (si Starter VM)
  8. Configurar Guacamole connection
  9. Crear grupos + políticas Netbird
  10. Enviar email bienvenida + credenciales
  11. ✅ Dashboard activo
ConnectExistingTSplusWorkflow (Plus)
  1. Crear org + tenant en Zitadel
  2. Generar Netbird setup-key para cliente
  3. Enviar instrucciones deploy nodo
  4. Esperar peer registration (webhook)
  5. Configurar ruta hacia TSplus del cliente
  6. Configurar NeoProxy (Pomerium) para TSplus
  7. Verificar conectividad ping TSplus
  8. Configurar token exchange SSO
  9. Crear grupos + políticas de acceso
  10. ✅ TSplus accesible vía NeoSC

  1. Crear org enterprise + B2B policy
  2. Habilitar delegación de usuarios
  3. Deploy nodo hardened en red cliente
  4. Configurar LDAP/AD federation

  6. Deploy NeoVault (JumpServer)

  8. Configurar VMs propias del cliente

# provision-windows-vm.ps1 # Ejecutado via WinRM desde FastAPI VM Service # Requiere: WinRM habilitado, PowerShell 5.1+ param( [string]$TSplusLicense, [string]$NetbirdSetupKey, [string]$OrgId, [string]$NeoscApiUrl = "https://api.kappa4.com" ) function Write-Step([string]$msg) { $body = @{org_id=$OrgId; step=$msg; ts=Get-Date -Format o} | ConvertTo-Json Invoke-RestMethod -Uri "$NeoscApiUrl/vms/progress" -Method POST -Body $body -ContentType 'application/json' Write-Host "[$(Get-Date -Format 'HH:mm:ss')] $msg" } # === PASO 1: Habilitar OpenSSH === Write-Step "Instalando OpenSSH Server..." Add-WindowsCapability -Online -Name OpenSSH.Server~~~~0.0.1.0 Start-Service sshd Set-Service -Name sshd -StartupType Automatic New-NetFirewallRule -Name sshd -DisplayName 'OpenSSH SSH Server' -Enabled True -Direction Inbound -Protocol TCP -Action Allow -LocalPort 22 # === PASO 2: Descargar e instalar TSplus === Write-Step "Descargando TSplus Remote Access..." $url = "https://dl-file.com/TSplus-RemoteSupport.exe" Invoke-WebRequest -Uri $url -OutFile "C:\Temp\TSplus.exe" Write-Step "Instalando TSplus (modo silencioso)..." Start-Process -FilePath "C:\Temp\TSplus.exe" -ArgumentList "/VERYSILENT /NORESTART /COMPONENTS=core,webserver,sts" -Wait # === PASO 3: Activar licencia === Write-Step "Activando licencia TSplus..." $tsplusPath = "C:\Program Files (x86)\TSplus\UserDesktop\files" $activateExe = Join-Path $tsplusPath "LicenseManager.exe" Start-Process -FilePath $activateExe -ArgumentList "/activate $TSplusLicense /silent" -Wait # Verificar activación Start-Sleep -Seconds 5 $status = Get-Content "C:\Program Files (x86)\TSplus\Clients\www\software\LicenseStatus.txt" if ($status -notmatch "active") { throw "TSplus activation failed" } # === PASO 4: Instalar Netbird === Write-Step "Instalando NeoMesh client..." $nbUrl = "https://pkgs.netbird.io/windows/x64/netbird_installer_latest_windows_amd64.exe" Invoke-WebRequest -Uri $nbUrl -OutFile "C:\Temp\netbird.exe" Start-Process -FilePath "C:\Temp\netbird.exe" -ArgumentList "/quiet" -Wait # Conectar al mesh con setup key del tenant Write-Step "Conectando VM al NeoMesh..." & "C:\Program Files\Netbird\netbird.exe" up ` --setup-key $NetbirdSetupKey ` --management-url "https://manager.kappa4.com" ` --hostname "neosc-vm-$OrgId" # === PASO 5: Reportar IP de Netbird === Start-Sleep -Seconds 10 $nbIp = (& "C:\Program Files\Netbird\netbird.exe" status | Select-String 'NetBird IP').ToString().Split()[-1] Write-Step "VM lista. NeoMesh IP: $nbIp" $confirm = @{org_id=$OrgId; netbird_ip=$nbIp; vm_status="ready"} | ConvertTo-Json Invoke-RestMethod -Uri "$NeoscApiUrl/vms/ready" -Method POST -Body $confirm -ContentType 'application/json'
